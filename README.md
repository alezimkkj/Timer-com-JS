⏱️ Cronômetro + Troca de Tema

Projeto simples desenvolvido com HTML, CSS e JavaScript puro, contendo:

⏳ Cronômetro funcional (start, pause, reset)

🌙 Alternância entre tema claro e escuro

🎨 Controle visual de botão ativo

💡 Manipulação dinâmica de classes no DOM

🚀 Funcionalidades
🕒 Cronômetro

Inicia a contagem em minutos e segundos

Pode pausar a qualquer momento

Pode reiniciar para 00:00

Evita múltiplos setInterval rodando ao mesmo tempo

🎨 Sistema de Tema

Alterna entre modo claro e modo escuro

Atualiza dinamicamente as classes do <body>

Mantém apenas um botão ativo por vez

🛠️ Tecnologias Utilizadas

HTML5

CSS3

JavaScript (Vanilla JS)

Font Awesome (ícones)

📚 Conceitos Praticados

Manipulação do DOM

addEventListener

classList.add() e classList.remove()

Controle de estado com variáveis

setInterval() e clearInterval()

Organização de código em funções

🎯 Objetivo do Projeto

Praticar lógica de programação e manipulação do DOM antes de avançar para frameworks como React.
